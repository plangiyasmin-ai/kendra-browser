Kendra Browser

Peramban Web Python GUI yang Cepat untuk Termux VNC.

Unduh dan Jalankan

Kendra Browser adalah aplikasi desktop Python. Anda perlu menjalankan *file* ini di lingkungan yang mendukung GUI, seperti VNC (Virtual Network Computing) di Termux.

Unduh kendra_browser.zip

Panduan Instalasi (Termux)

1. Pasang Prasyarat (Di Termux Biasa):

$ pkg update && pkg upgrade
$ pkg install python qt5-qtwebengine python-pyqtwebengine

2. Ekstrak dan Jalankan (Di Sesi VNC/XServer):

# Ekstrak file yang diunduh
$ unzip kendra_browser.zip
# Jalankan aplikasi
$ python kendra_browser.py
